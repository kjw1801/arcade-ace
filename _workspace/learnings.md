## 0. 작업 인프라 규칙 (프로세스)

- **단일 진실원본 = git.** 작업은 워크트리 브랜치(`claude/...`)에서 하고 **회차 단위로 커밋 → 메인 fast-forward**.
  - ✗ 실수: 메인↔워크트리를 `cp`로 수동 복사 → 드리프트·회귀(PROJECT_CONTEXT 옛 규칙 잔존 등). **cp 동기화 금지, git으로 합친다.**
- **설정 드리프트 점검:** 회차 시작 시 CLAUDE.md / PROJECT_CONTEXT / novel-config / platform-guide / canon-bible 정합 확인(특히 POV 규칙·코드명·인물 역할).
- `.gitignore`로 .DS_Store 등 노이즈 제외. 스크래치(EP00x_*.md)는 `manuscript/_archive/`에만.
- **블라인드 자가-검증 테스트(스킬 QA):** 방법론(config+learnings)만으로 회차를 백지 재생성해 스킬 빈틈을 찾는다. **반드시 골드/이전본을 working tree에서 *물리적으로 제거* 후 실행**(지시만으론 에이전트가 Read해 오염됨 — v2 사례). 루프: 블라인드 재생성 → 갭 발견 → learnings 패치 → 물리 블라인드 재검증. (EP001: v1→갭(엔진·회상)→규칙10·11→v3 해소 확인.)
  - 추가 주의: 재생성 에이전트가 원작 파일 끝의 종료 `***`를 따라 붙이는 경향 → 규칙 6(`***`=실제 전환 1회 전용, 종료 마커 금지) 강제.

## 1. EP001에서 도출한 재발 방지 규칙

| # | 규칙 | EP001 실수 사례 |
|---|---|---|
| 1 | **원작 대사 잠금** (novel-config SOP-A①). 검증된 원작 대사는 그대로. 설정·연속성 충돌·명백한 어색함일 때만 최소 수정. | "나는 이미 제임스 본드다" → "진작에 제임스 본드 됐다"(번역투) / "그냥 머슴이지" → "인건비 굳지, 24시간 콜, 만능 머슴"(과적재) |
| 2 | **추가 개그·정보형 대사가 기존 펀치와 경쟁하면 기각.** | "500원에 평생을 걸 거냐고"가 진짜 펀치 '경위서 쓰는 귀찮음'을 약화 |
| 3 | **서술 확장 ≠ 대사 복원** (별개 판단). 서술 질감은 늘려도 대사는 원작 말맛 우선. | 빈소 서술은 복원·확장 OK였으나, 그 김에 대사까지 재작성한 게 잘못 |
| 4 | **시간 구조 명시.** 콜드오픈/회상 시 *되감기 신호* + *경과 신호*. 인물 동선과 시간 일치. | 스콜피온이 "출구 2걸음"을 "두 시간"에 걸쳐 걷는 모순 → 되감기("시곗바늘을 몇 시간 되감는다") + 경과("어느덧 오후 다섯 시") + 연속화로 해결 |
| 5 | **교차 중복 점검.** 직전화 + 원작 통독 후, 시그니처 묘사·셀레브레이션·목록의 회차 간 반복 차단. **직전화가 잠겼으면 새 화를 변주.** | EP001 라스트 "3주간 비웃었던…박자/레이싱/신발 뒤축" ↔ EP002 둘째 단락 거의 동일 |
| 6 | **`***`는 실제 장소·시간·장면 전환만.** 동일 장면 초점 이동(외부 오독 포함)은 무구분(전지적 기반). | EP001 컷 3→1, EP002 5→3 목표 |
| 7 | **de-gild(멋 빼기):** 동어반복(식은땀/식어), 추상·과시 비유(손끝에 담는 영역), 감정명사 명명(공포를 느꼈다), 상황주어 사역(무시가 확신을 건드렸다) 지양. **평문 FID 유지.** | "몸에 새겨진 가난의 무늬"·"손끝이 먼저 부정했다"·"통째로 얼어붙게 만든" 등 삭제·담백화 |
| 8 | **포맷·잠금:** 내적독백 평문(별표·이탤릭·작은따옴표 독백 ✕) / 마스터떡밥(누나·No.0·Admin-L·BlackArray) 비노출, JW-0·고스트는 코드명 수준만 / 분량 부호제외 3,000+ (3,500 권장, **오버는 결함 아님**). | — |
| 9 | **카드 분배:** 폭발물 비유·FMC '명명'·T-1 격상·둘째 뽑기 = **EP002+ 전용**. EP001엔 'elite/1급 영역' 인식까지만. | — |
| 10 | **HOOK1K 오독 엔진(필수).** '갭이 있다'로 끝내지 말 것 — 정우의 평범 행동을 외부가 **첩보로 오역하는 구체적 '오독 번역' 비트**를 첫 1,000자에 1개+ (정본: 코골이→'모스부호 송신', 식중독→'독극물 내성 훈련'). 원작에 없어도 창작해 `misread-state.md`에 회차별 비축. | 방법론 블라인드 재생성이 일반 갭(99%↔몸치)만 넣고 이 엔진을 놓침 |

## 2. 회차 체크리스트 (집필/퇴고마다)

### 집필 전 (입력 통독)
- [ ] **직전 EP 통독** — 끝 비트·클리프행어·연결 훅 확인 (continuity)
- [ ] **원작 해당화 txt 통독** (`source-50/per-episode/`) — 대사·비트 원천 확보 (대사 잠금 기준)
- [ ] **misread-state.md** 해당화 오독 상태(누가 정우를 어디까지) 확인
- [ ] **canon-bible** 수치·등급·연표 교차
- [ ] 직전화와 **중복될 시그니처 묘사 목록업** (관성락 원리·콜라캔·비유·인물 텔 등)

### 집필 후 (검수 — novel-config 12축 + 아래)
- [ ] **원작 대사 잠금** — 가필/덧칠 점검 (규칙 1~3)
- [ ] **시간선·동선 정합** — 되감기/경과 신호, 인물 위치-시간 일치 (규칙 4)
- [ ] **교차 중복** — 직전화 대비 반복 제거 (규칙 5)
- [ ] **`***` = 실제 전환만 / 별표 0** (규칙 6)
- [ ] **de-gild** 패스 — 동어반복·추상비유·감정명사·상황주어 (규칙 7)
- [ ] **마스터떡밥 비노출 / 카드 분배 / 분량 / FID 평문** (규칙 8~9)
- [ ] **HOOK1K 오독 엔진** 첫 1,000자에 1개+ / **회상 markerless**(―·--- 금지) (규칙 10~11)
- [ ] **양쪽 동일 + 회차 커밋** (프로세스 0)
