# misread-state.md — 착각(오독) 상태표

> 원작 50화 + 외전 3편 역추출. **누가 / 정우의 어떤 평범한 행동을 / 무슨 첩보 능력으로
> 오해했는지 / 어떻게 심화·확산됐는지**의 타임라인. MISREAD 커스텀 축의 근거 문서.
> 리부트 집필 시 "이 인물이 이 시점에 무엇을 알고/모르고 있어야 하는가"의 단일 진실원본.

---

## 0. 오독 메커니즘 3대 법칙 (작품 공식)

1. **관성 락** — 정우의 진짜 능력(오락실/엔지니어)은 항상 첩보 능력으로 1:1 번역된다.
   정우 본인은 끝까지 자기를 평범하다고 믿는다.
2. **비가역성** — 오독은 회수 지점 전까지 풀리지 않는다. 정우의 부정·선 긋기 시도는
   **전부 역효과**로 작동해 오해를 더 굳힌다 (28화 "오해가 콘크리트처럼 굳어버렸다").
3. **진실 강화의 역설** — 진실을 말할수록 신화가 강해진다. (40화: "그냥 평범한 동네
   수리 기사"라는 진실 댓글이 "현직 냄새"로 역해석됨 / 50화: 천장 붕괴라는 우연이
   오독을 '증명'함.)

## 1. 오독 확산 단계 (전체 궤적)

```
스콜피온(1화, 외부 최초 발화)
   └→ 정하린(3화, 국정원 내부 점화 + "선배님" 호칭)
        └→ 김주환(4→16화, 의심→신앙)
        └→ 강무영/국정원장(6·9·26·40·41화, 빈칸·침묵을 신화로)
             └→ 대통령 라인(7·26화, 진짜 배우 '이정우' 잡담을 작전 암호로 역해석)
   └→ 업계 전체(31화, "오락실 불가침" 정설 = 6명 '증발'을 정우가 '삭제')
        └→ 강태오/JSICS(36~38화, "판을 짠 쪽"으로 추적 대상화)
challenge 확산: 유진/커뮤니티(39~42화) → 실검 1위·밈·글로벌(40화) → 미국 공식 문의(40화)
가족 오독: 유나(29·30·42화, 정하린=연인/임신/동거로)
정치 오독: 박태산(48~50화, 첩보 가문→목숨 빚)
신화의 외부 원본: 외전1~3 (JW-0=Zero=Ghost 괴담 스펙 + "선배님" 호칭의 기원)
```

## 2. 인물별 오독 상태 (현재 = 50화 종료 시점)

| 인물 | 정우를 무엇으로 아는가 | 오독 강도 | 회수 여부 |
|---|---|:--:|:--:|
| 정하린 | 코드명 Zero, 교본 밖에서 성립한 전설적 요원("선배님") | ★★★★★ | 미회수 |
| 김주환 | 모든 행동이 교본인 살아있는 전설 | ★★★★★ | 미회수 |
| 스콜피온 | 측정 불능의 괴물 '제로'(외전: JW-0=Ghost 괴담) | ★★★★★ | 미회수 |
| 강무영 | 침묵으로 답하는 정체불명 최상위 자산 | ★★★★☆ | 미회수 |
| 강태오(적) | 이 판을 설계한 쪽, 1순위 추적 대상 | ★★★★☆ | 미회수 |
| 박태산 | 내 라인을 꿰뚫은 첩보 조직 인물 + 생명의 은인 | ★★★★☆ | 미회수(역설 강화) |
| 유진/커뮤니티 | '선배님' = 전설의 블랙 요원(밈·실검 1위) | ★★★★★ | 미회수 |
| 유나 | (첩보는 모름) 정하린과 사귀는/동거하는 삼촌 | ★★★☆☆ | 미회수(가족계) |
| 춘식 | 자길 거둬준 무서운 '주임님' | ★★★☆☆ | 미회수 |
| 일반 대중/미국 | 정체불명 'Senior'/Ghost | ★★★★☆ | 미회수 |
| **정우 본인** | **그냥 평범한 7급 시설관리직** | — | **관성 락(영구)** |

> ⚠️ **전 구간 통틀어 오독이 '풀리는' 명시적 회수 장면은 없다.** 가장 회수에 가까운
> 50화조차 우연(천장 붕괴)이 오독을 강화하는 방향으로 귀결된다. 리부트에서도 이
> 비가역성을 유지(guard_rail #2).

## 3. 화별 오독 비트 로그

### 1부 (1~13화) — 점화
- **1화** 스콜피온: 인형뽑기 '관성 락' → **미세 근육 제어술(FMC)**. 몸치·레이싱 충돌 →
  '평범 실패자를 완벽 연기하는 위장'. (생체 리듬이 JW-0 후보군과 99% 일치 첩보가 전제)
- **2화** 스콜피온: 두 번째 곰인형 성공 → **T-1급 테크니션** 판정 격상.
- **3화** ★최초 점화: 칼퇴 1초 전 방해 손을 쳐낸 게 출입증 모서리로 스콜피온 이마 타격
  + 18:00:00 정각 태그 동시 완수 → 정하린이 '3주 기만전술의 완성, 외과수술적 정밀 타격'
  으로 해석 → **첫 "선배님" 호칭**. 동시에 코드 레드 봉쇄와 겹쳐 '함정의 설계자'로 오해.
- **4화** 스콜피온: 이마만 긁힌 것(그냥 쳐낸 것) → '윗선까지 끌어올리려 일부러 살려둔 계산'.
- **6화** 스콜피온의 3주 관찰일지(집대성): 코골이 낮잠→도시형 위장 보법/모스부호, 식중독
  설사→독극물 내성 훈련, 중학생에게 500원→민간인 앞 위장 연기. / 강무영: 부처 간 칸막이로
  '소속 불명' 상태 → **관료주의 은신(Bureaucratic Stealth)**.
- **7화** 정하린: 조카와 논 것→'가족 커버 루틴'. / 강무영: 대통령의 "공익 광고·국민배우
  섭외" 지시(진짜 배우 이정우 얘기)를 '음지의 칼을 양지로 세우라는 작전 승인 암호'로 이중 오독.
- **9화** 스마트워치 SYNC ERROR에 정우가 '고장 변상 공포'로 심박 162 → 정하린/강무영이
  '방호 프로토콜을 감지하고 전투 임계점까지 예열'로 오독. 정우의 "오락실 기계 교체팀"
  실언 → **커버네임으로 공식 채택**(조직명 기원).
- **11화** 정우가 펌프 기계 속 P-03 통신모듈을 '불법 튜닝'으로 분해 → '우리 흔적을 다
  알고 작전 출발점을 재세팅'.
- **12화** 콘덴서 진단을 '500원 동전 잡아먹는 하마'로 표현 → 정하린 '동전으로 말해주시는'
  으로 오독('하마의 경제학').
- **13화** 동전 교환기 '콜렉터 잼' 표현 → 정하린이 '자금/정보 수집 노드 + 통신 방해'
  데드 드롭으로 의심. "서비스룸이 시작점" → '구역의 심장·작전 원점'.

### 2부 (14~26화) — 내부 심화 + '임연수' 떡밥
- **14화** 칼을 안 피하고 발밑 전선부터 본 것(합선 걱정) → '킬존 지형 장악'. 메인 차단기
  내림(불 끄려고) → '물리 계층 하드 킬'.
- **15화** "전력 많이 먹는 거 같이 돌리지 마세요" → '숨을 나눠 들키지 않게'. 멀티탭 과부하
  서버 셧다운 → **고도 보안 '부비트랩'**.
- **17화** 긴축으로 캔커피 안 산 것 → '흔적 줄이는 패턴 변화'. "식후에 제로 콜라 한 캔" →
  작전 암호("점심. 식후. 제로").
- **18화** 작업등(투광기) 스위치 → '광원부터 현장 장악'. 덜 마른 왁스/고장 승강기 경고 →
  '적을 미끄러운 구역으로 유도, 퇴로 차단'.
- **19화** ★떡밥 점화: 싫어하는 생선 "**임연수**" 언급 → 정하린이 '가시·쓴맛=미확인 변수'로
  수첩 기록. 24·26화까지 거대 오독으로 발전.
- **20화** 부품함 강철 구슬을 엎어 덩치를 미끄러뜨림 → '중력·마찰만으로 무장 적 무력화한
  완벽한 아날로그 트랩'.
- **23화** 자살하려는 스콜피온을 막으려 사원증 끈을 휘둘러 이마 타격 → 스콜피온이 '굴욕적
  태그'로 받아들이고 붕괴("때린 데를 또 때려?… 엄마").
- **24화** 측은지심 "오락실 알바라도?" → 정하린 '위험 자산 포섭'. 차에서 잠든 것(피로) →
  '피로가 아니라 통제'.
- **26화** ★윗선 확산: 강무영이 대통령 잡담("이정우 또 나왔네, 연기 잘해, 공익 광고에
  세워")을 '음지의 칼을 양지로'로 역해석. 정하린이 현욱이 보낸 구내식당 메뉴표 '임연수
  구이'를 '관리책 확인·접선·작전 개시 신호'로 번역.

### 3부 (27~39화) — 가족 오독 + 적 빌런 마킹 + 밈 폭발
- **27화** ★표본 비트: 배고파 비틀대는 걸음 → '위장 보행'. "제육 털어버려야지" 통화 →
  **'J-6 내부 소탕 작전'**("제육"→"J-6"). 구내식당 전력질주 → '제로가 뛰는 작전 구역'.
- **28화** 정우의 진심 어린 선 긋기("저 평범한 시설관리직")→ 정하린 '위장(Cover)의 기본을
  강조하는 꾸지람'으로 오독 → 오해가 콘크리트처럼 굳음.
- **29화** ★가족 시트콤 정점: 유나가 정우↔정하린 언쟁을 '책임 안 지려는 나쁜 남자'로,
  "임무 4주 차"→'교제 4주', 차/담배 냄새 헛구역질→**임신**으로 확정.
- **30화** 정우가 JSICS 감시조에게 "스토킹 신고한다" 막아선 것 → 유나 '가족 지키는 상남자',
  업무 발언("내 사람 건드리면 못 참아")→'사랑 고백'.
- **31화** 업계 전체: 오락실에서 6명 '증발' → '정우가 삭제'("오락실 불가침" 정설). C-11·
  스콜피온이 진짜 낡은 작업복→'완벽한 잠복 위장'("생활조차 커버").
- **32화** 보안요원 신원조회 위기에 정우가 귀찮아 "오기 싫었거든요" 돌아섬 → '이탈 선언으로
  상대 결정을 강제, 물리력 없이 문 열게 한 심리전'.
- **36~38화** ★빌런 마킹: 화재 시 직업 본능으로 대피 유도("한 줄로! 손잡이 잡고!") →
  강태오가 '난장판을 설계하고 유유히 걸어나오는 판의 주인'으로 오독(36화). 김주환의
  "선배님" 호칭 + NIS가 정우 주변을 비우는 동선 → 강태오 결론 "저게 판을 짠 쪽"(37화) →
  **정우를 1순위 추적 대상화**, 박재식에게 보고(38화).
- **39화** ★밈 폭발: 헤드셋 줄 끊으며 "그거 버려. 걸려 넘어져"(안전 지시) → 유진이 '구속을
  버리고 일어나라는 인도자의 메시지'로 커뮤니티 박제 → 실검 1위 "선배님", 짤·릴스·렉카 확산.

### 4부 (40~50화) — 사회 현상화 + 정치 오독 + 역설적 '증명'
- **40화** ★스케일 폭발: "걸려 넘어져 챌린지"가 K-팝 아이돌 안무·글로벌 밈으로. 미국이
  "Who is the 'Senior'? Request contact." 공식 문의. 진실 댓글이 '현직 냄새'로 역강화.
- **41화** 강무영이 정하린의 빈칸 보고("특이사항: 없음")를 '나까지 선 안에 못 들어오게
  비우게 만든 고도의 보안 판단=훈장'으로 오독. 밈 "그거 버려"를 박태산 정치 최후통첩으로 전용.
- **45~46화** 현욱 대타로 떠맡은 이천 물류센터 점검 공문(특근 수당 목적) → 정하린 '경유
  부서 없음=흔적 줄여 선배님을 밖으로 꺼내려는 의도'. SUV 끼어들기→미행.
- **47화** ★표본 비트: "돈까스 맛있으면 체크리스트가 짧아지거든요"(식욕) → 김주환·정하린이
  '보급선 파악·심리전·정보 수집'으로 오독. *(명시적 진실 서술: "정우는 진짜 돈까스가
  기대됐을 뿐이었다.")*
- **48화** 발바닥 진동·소리로 트러스 이상 감지(엔지니어 직감, 사실상 진짜 실력) → '육안
  불가 영역의 현장 감각'. "안 돼!"로 지사 시찰 몸으로 막음 → 클라이맥스.
- **48~50화** ★박태산 오독→역설적 증명: 떨며 한 아부("지당하십니다")와 경고를 처음엔
  '겁먹은 7급'으로 봤으나, 정우→정하린→김주환의 '경고 후 예우 전환' 순서를 '훈련된 조직'
  으로 오독. **50화 천장이 박태산 섰던 자리에 실제 붕괴** → "그 사람이 나를 살렸다 =
  일정·건물까지 알았다 = 내 라인이 뚫렸다"로 재오독(우연이 오독을 '증명').

### 외전 (51~53화) — 신화의 외부 원본
- **외전1·2(스콜피온)**: 정우 미등장. 그러나 **JW-0 = Zero(No.0) = Ghost**가 외국 정보조직
  공식 신화로 존재함을 보여줌. "사람이 아니라 패턴이다. FMC 기반. 0.2초 반응. 모든 축에서
  측정 불능." 스콜피온만 "출처가 뭡니까" 의문 → 상관 "네 등급이 알 일 아니다". 한국이
  'Ghost의 나라'로 신화화. → 오독의 출처가 권력에 의해 의도적으로 흐려지는 메타 구조.
- **외전3(정하린)**: "선배님" 호칭의 기원 회수. 교육센터 시뮬레이션 Zero 패턴 케이스(아이를
  먼저 빼내고도 임무 완수, 교본 밖에서 성립)에 매료. 정하린은 교본대로 아이 포기(합격),
  Zero는 47초 지연·부상·로그 손실 감수하고 둘 다 살림. 미제출 노트 "…언젠가 실제로
  만나게 된다면, 그땐 '선배님'이라고 부르자." (정우=Zero 오독 전제의 정서적 완성)
