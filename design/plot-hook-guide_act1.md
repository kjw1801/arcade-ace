## 0. 1권 = 스콜피온 편 (자산 풀과 1막 경계)

- **1권 범위:** 로드맵 기준 **1~35화(스콜피온 편)** — 원작 1~27화(스콜피온/신화화 arc)를 소스로 확장 + 양아치편 신규. (원작 50화 ≠ 1권: 원작 후반·외전은 2·5권 소스로 분산. 정본 = §1-0 로드맵 매핑.)
- **1권 코어 아크:** 스콜피온 빌런 → 괴담 제조기 → 오퍼레이터에게 배신·정리당함 → 정우의 "오락실 알바라도?" 제안으로 **조력자 이동**(완치 아님, 방향 이동). 강태오·밈·박태산은 1권 후반 확장/2권 도약대.
- **1막(본 가이드 상세 범위) = EP001~005.** 원작 1~5화에 근사 1:1 대응. **노피아 선작 골든 구간**(EP001 선작 사활).
- **1권 엔딩 마커:** [ID 충돌: JW-0 vs JW-0007] 로그 (canon §4). 원작 본문엔 명시 로그 없음 → **리부트가 명시 카드로 신설 권장**(8화 분기 → 1권 클로징). 2권+ 진실(누나 Admin-L 더미 데이터) 회수 동력.
- **불변 가드레일(전 회차):** 관성 락 영구(정우 본인 오독 끝까지 미회수) / 오독 비가역 / 유나 로맨스 분리(코미디·정서 축) / 마스터 떡밥(누나·No.0·Admin-L·BlackArray) 1권 비노출(코드명·괴담 수준만, 3~5권 회수) / 정우↔정하린 슬로우번 씨앗만.

## 1. 리부트 페이싱 원칙

### 1-0. 마스터 구조 = 300화·8권 로드맵 (페이싱 정합) ★정본

> 매크로 정본: `기존 설정집/전체 시리즈 로드맵.md`(v0.7, 300화). **이 로드맵이 마스터.**
> 원작 본편 50화 + 외전 3편은 **소스 자산**(장면·오독·명대사·캐릭터) — *챕터 1:1 템플릿이 아니다.*
> EP↔원작 1:1 대응은 **골든런(EP001~005 = 원작 1~5화) 한정.** 그 이후는 로드맵 권/비트가 우선이고, 원작은 keep-scenes·명대사집·misread-state로 끌어온다. **3~8권은 대부분 신규 확장.**

|---|---|---|---|---|
| 1 | 1~35 | 스콜피온 | 원작 1~27(인형뽑기·관성락·신화화·코드네임 제로) + 양아치편 신규 | 전설의 시작. **EP001~005 = 1권 도입(골든런)** |
| 2 | 36~70 | 정하린 | 원작 일부 + 신규(총알 시퀀스·유나 첫 대면) | 보좌하는 자. '선배님' 심화 |

- **강태오/JSICS 배치(canon §6 '1부 빌런' 정합):** 권별 스파인은 트라우마 인물, 강태오/JSICS는 그와 *병행하는 외부 위협 라인*. 1권 후반(원작 36화 "재밌네" 마킹·38화 '선배님' 상부 전파) 점화 → 1권 말~2권 외곽 추적(신작 데모 EP051·052가 이 라인) → 6~8권 외부 세력(셧다운) 합류. 각 권의 작전/현장 갈등을 공급하는 적이고, 정서 스파인은 그 권의 트라우마 인물.
- **마스터떡밥 회수 일정:** 누나 Admin-L/No.0/BlackArray 진실(=99% 일치 기원)은 **5권 '과거의 그림자'에서 회수 시작**. 1~4권은 코드명·괴담 수준만(guard_rail #9). (이전 메모 "3~5권 회수"의 *착수 지점 = 5권*으로 확정.)

### 1-1. 원작 ↔ EP 대응 (초반 근사 1:1 — 골든런 한정)

| EP | 원작 대응화 | 대응 유형 | 비고 |
|---|---|---|---|
| EP001 | 원작 1화 | 1:1 (LOCKED) | 빈소 회상은 **페이오프 직전**으로 재배치(원작은 콜드오픈). HOOK1K 오독엔진 3비트로 도입 강화. |
| EP002 | 원작 2화 | 1:1 (페이커 회수됨) | 둘째 인형(FMC 명명) + T-1 격상 + 칼퇴 1분 30초 + 접촉식 검증 직전 컷. 페이커는 EP004로 이월. |
| EP003 | 원작 3화 | 1:1 | 18:00 정각 출입증 모서리 타격 + 동시 태그('0초 클리어'). 코드 레드. **첫 '선배님'(정하린 등장)**. |
| EP004 | 원작 4화 | 1:1 (압축 주의) | 조사실/외교부 면책/지하주차장 스콜피온 정체 완전 인지(**페이커**)/정하린 신격화·원장 보고. 4개 장면 → 노피아 band 압축. |
| EP005 | 원작 5화 | 1:1 | 관사 귀가·유나 통화(거짓 설명)·공무원식 경위서·"싹 다 갈아엎어야지"→감찰실 '선전포고' 이중해석. |

> **압축 구간 표시:** EP004가 1막 유일한 **고밀도 압축 후보**다. 원작 4화는 (조사실 심문)→(외교부 면책)→(지하주차장 스콜피온)→(정하린 30층 보고) 4블록. 노피아 band(부호제외 3,000~4,500) 안에 4블록을 다 넣으면 밀도 폭증·이탈 위험(learnings 규칙 13: 외부 오독 = 압축 FID). **외교부 면책 블록(HOOK1K)**과 **지하주차장 페이커 블록(라스트 훅)**을 양 기둥으로 고정하고, 조사실은 면책으로 무너지는 도입부로 흡수, 30층 보고는 라스트 1컷으로 압축한다. 분량 초과 시 EP004를 2화로 분할(EP004a 조사실·면책 / EP004b 페이커·신격화)하되, **1막 페이싱 1:1 원칙상 분할은 최후 수단**.

### 1-2. 분량 band (novel-config SOP-B⑨)

- **회차당 부호 제외 3,000자 이상**(유료 플러스 최소). 권장 **3,000~3,500**(부호 제외) ≈ 공백포함 약 4,300~5,100.
- 1막 골든 구간은 **band 상단(3,300~4,500)** 허용 — EP001 LOCKED가 부호제외 ~4,300으로 레퍼런스. 오버는 결함 아님.
- 측정: 회차마다 글자수 스크립트로 '부호 제외' 확인 → 3,000 미달 시 **원작 디테일 복원**으로 보강(신규 창작 ✕).
- 조회수 기반 정산(구독제) → 한 편 길게 빼기보다 band로 **꾸준히**.

### 1-3. 컷·시점 규율 (learnings 규칙 6·13 / novel-config 시점·포맷)

- `***` = **실제 장소·시간·장면 전환 1회만**. 동일 장면 초점 이동(정우↔유나, 정우 행동↔스콜피온 오독 반응)은 **무구분**(전지적 기반). EP001 = `***` 1개, EP002 = `***` 목표 3개.
- 외부 오독 인물이 **별도 장소·시간의 새 장면**을 열 때만 `***` + 첫 문장에 인물 이름 박기.
- 외부 오독 전달 = **압축 FID + 1줄 츳코미**. 본부 무전 『첫째·둘째·셋째』 연속 대사 ✕(모바일 밀도 폭증). 80자+ 장문 금지.
- 회상/플래시백 = **markerless**(시간 앵커 진입·복귀, `―`·`---`·별도 `***` 금지).
- 라스트 훅 = **3층 다층화**(기술+정서+외부충격) + 명시적 이탈방지 1줄(learnings 규칙 14).

## 2. EP001~005 상세 카드

> 각 카드 = {원작 대응 · 핵심 비트 · HOOK1K · 라스트 훅 · 들어갈 명대사(배치) · 오독 상태 진행 · 카드 분배(노출/비노출)}.

### EP001 — 〈국가권력급 요원은 인형을 뽑는다〉 · 원작 1화 · **상태: LOCKED**

- **핵심 비트:** 콜드오픈(스콜피온 감시 3주 차, 99% 일치 첩보 깔기) → 오후 2시 로비(유나 방문·마스터키 거부) → 펌프/레이싱 몸치 → 인형뽑기 직전 "혼자 있어도 안 무서울" → **빈소 회상**(유나 고아화·친척 분쟁·이어폰·"밥 안 굶긴다") → **관성 락 발동**(0.2초 정밀 / 골든 리트리버 1발) → 스콜피온 발 묶임("철수 미뤄졌다").
- **HOOK1K(오독엔진 3비트, learnings 규칙 10):** 코골이→'모스부호 송신' / 식중독→'독극물 내성 컨디셔닝' / 동전 쥐여줌→'가족 커버 위장 연기'. 각 비트에 스콜피온 deflation 츳코미('침 흘리는 낮잠'·'배탈이라고'·'진짜 삼촌인데'). + 99% 일치 첩보 전제. **※ 3비트 = 원작 6화 관찰일지 *메커니즘*의 조기 배치(규칙17 예외 — 장면 시그니처 비복제). EP006(원작 6화)은 음성메모·인이어 통신 로그로 심화·차별화(중복 방지).**
- **라스트 훅(→EP002):** 기술(0.2초 관성 락 목격) + 정서(유나 환호) + 외부충격(콜라 캔 우그러짐·스콜피온 발 안 떨어짐) 3층 + 명시 이탈방지("철수는, 잠시 미뤄졌다") + 정우 곰인형 독백(둘째 뽑기 연결).
- **명대사:** "공무원이 제일 무서워하는 게 뭔지 아냐? …경위서 쓰는 귀찮음이야."(정우) / "저거 있으면… 집에 혼자 있어도 안 무서울 것 같아."(유나) / "근데 안 짤려. 철밥통이거든."(정우, 빈소) / "삼촌이… 기계의 '꼼수'를, 실력으로 박살 내 줄게."(정우).
- **오독 진행(스콜피온):** 1단계 심각한 몸치 → 2단계 생활형 텔 관찰(신발 뒤축·굳은살·윤활유) → 3단계 '그냥 삶 그 자체'(일반 공무원 판정) → 4단계 인형뽑기 직전 눈빛 변화 감지 → 5단계 관성 락 목격, 인식 전환 임박('1급 요원이나 가능한 정밀함').
- **카드 분배:**
  - **노출:** 코드명 '고스트(Ghost)'·'JW-0'(코드명 수준만) / 99% 일치 첩보(마스터 떡밥 **씨앗**, 진짜 기원 미노출) / 관성 락 명명(정우 자칭) / 유나 14세 설정·일상 감탄('능력자네' 수준 — **FMC 정식 명명은 ✕, EP002+**).
  - **비노출:** FMC '명명'·T-1 격상·둘째 뽑기(=EP002 전용, learnings 규칙 9) / 선배님·정하린 / 폭발물 비유 / 마스터 떡밥 진실(누나·No.0·Admin-L·BlackArray). 스콜피온 reaction은 '1급 요원 영역'까지만.

### EP002 — 〈0.2초의 승부사〉 · 원작 2화 · **상태: 페이커 회수됨(정합 진행)**

- **핵심 비트:** 둘째 곰인형(벽 인형·난이도↑) 성공 → 스콜피온 FMC 단정·T-1 격상 본부 보고 → 4시간 체류·철권 단발머리 패배 → 칼퇴 1분 30초 박박이 → 유나 안전 배웅 루틴(조카=약점 확인) → 태그기 직전 스콜피온 균형 흔들기 → **접촉식 검증(경동맥 2cm 옆 신경 다발) 손끝이 목덜미 닿기 직전 컷**.
- **HOOK1K:** 첫 인형 '우연이겠지' 미련 → 둘째 동전 투입 → 벽 인형(확률 조작형, 시스템이 가장 단단히 잠그는 변수)을 0.2초 관성으로 뚫음. (스콜피온 오독 시점 도입 — 압축 FID.)
