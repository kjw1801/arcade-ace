## 0. 작품 대전제 (절대 불변)

- **정우는 끝까지 일반인.** 초인 각성·"사실 나는 강했다" 자각 **금지**. 동기는 언제나
  칼퇴 / 500원 / 징계·경위서 회피 / 유나·"내 사람" 챙기기.
- 멋있음은 **항상 타인의 해석에서만** 발생. 내레이션은 절대 "정우=초인/설계자"로 단정
  금지 → "~처럼 보였다 / ~라고 판단했다 / ~라고 믿었다"로 처리.
- **진짜 괴물은 초능력자가 아니라 정권·관료제·정보기관·외교라는 시스템.** Ghost 신화의
  실체 = 일반인 + 시스템 병크 + 누나의 마지막 꼼수.
- **위기 반응 규칙**: 평소엔 찌질한 생활인이지만 "내 사람"이 죽을 것 같으면 **머리보다
  몸이 먼저 나간다**(2권 총알 시퀀스의 근거).

## 1. 요원 등급 체계 (NI-ATC-21-04, 한국 국정원 로컬 버전)

3축 비공식 분류 — **S(Striker/전투)·W(Watcher/정보)·T(Technician/장비)**.

| 축 | 등급(낮음→최상위) | 비고 |
|---|---|---|
| S (Striker) | S-3 기본 → S-2 팀장·실전 → **S-1 대테러 최정예** | 공식 문서는 보통 S-2까지만 운용 등급. S-1은 "교육센터/극비 자료의 상징 등급" |
| W (Watcher) | W-3 기본 감시 → W-2 지역 패턴 → W-1 다국적 종합분석 | |
| T (Technician) | T-3 기본 → T-2 고난도·폭발물 → **T-1 희귀급 정밀(세계 극소수)** | **FMC(미세 근육 제어술)** 보유가 T-1의 상징 |

- **"보수적 스펙" 메타 룰**: 한국은 정치·시험환경 탓에 사람·무기 스펙을 **항상 낮게
  적는** 나라. 실전 S-1급도 문서엔 S-2. → 2권 중간보스 유언("문서엔 S-2라더니 이건
  S-1 상위급… 사람 스펙도 이렇게 적냐") + 방산 스펙 개그로 회수.
- **Ghost/Zero는 등급이 아니라 프로젝트/패턴 코드명.** "Ghost = S-1++" 같은 등가
  취급은 전부 오해/과대해석.

## 2. JW-0 / No.0 / Ghost / BlackArray — 신화의 진짜 정체

- **JW-0 (Project Zero)**: 국정원 극비 프로젝트. 목표는 **사람이 아니라 패턴/데이터**
  (FMC·생체리듬·0.2초 반응 로그) 확보. 결과: 완전일치 0명, 부분일치 3명, **98%+ 일치
  No.0 패턴 단 1개**. No.0의 실명은 전면 말소 → "패턴에 붙은 코드"일 뿐.
- **폐기**: 성과 미달·예산 누수·문서 조작 의혹·해외 압박으로 폐기, 전 기록은 암호화
  저장 풀 **BlackArray**로 이관. 공식상 JW-0는 "없었던 프로젝트".
- **Ghost**: 국정원 내부 은어 / 해외선 JW-0 No.0 패턴 코드. 국적 불문 "괴담의 이름".
  "교전 후 생존자는 없다 — 있다면 배신자뿐" 괴담이 따라붙음(오퍼레이터가 신앙).
- **정우가 No.0와 99% 일치로 찍힌 진짜 이유** ↓ (작품 최대 반전, 3~5권 회수)

### ★ 오독의 진짜 기원 = 누나(Admin-L/Angel)
- 정우 누나(=유나 엄마)는 과거 **JW-0 데이터 관리자 Admin-L**(별칭 **Angel** = "로그
  위의 천사", 기록을 지우고 끝내 아이들을 시스템 밖으로 빼내려 한 인물).
- 매형(국정원 해외라인 요원)이 정권교체·개혁 와중 **정치적 희생양으로 작전사망**하자,
  누나가 BlackArray의 JW-0 데이터를 **교란**: No.0 빈 슬롯에 **정우의 군대·알바·청년기
  패턴을 더미로** 채워 넣고, 인사·예산 라인에 꼼수를 걸어 **"어느 부처도 책임 안 지는"
  상태**를 만든다. 의도: "동생은 민간인이니 시스템이 쓸모없는 샘플로 보고 버리겠지."
- **역효과**: 이후 정권교체·전산 이관 병목과 겹쳐 → ① 정우 더미 패턴이 오히려 No.0와
  **99% 일치로 격상**, ② 인사 항목이 부처 사이를 떠돌다 **Grey Zone(행정적 은신)** 으로
  굳음. = 강무영이 "부처 칸막이를 역이용한 초정밀 위장"으로 과대해석하는 그것.
- **행정적 은신 = 누나의 보호 시도 20~30% + 관료제 병크 70~80%.** 강무영의 "정밀 설계"
  이미지는 **100% 과대해석**.
- 정우·유나는 끝까지 "빚/사고/보험금 5억"만 안다.

### ARCADE / 오퍼레이터
- **ARCADE 서버**(B구역 비인가): 국정원 내부망 대역(192.168.77.x) 경유 → 해외 IP·
  다국적 브로커 네트워크로 연결(구체 연결은 후반 회수). 검은 토큰 = 암호화 데이터
  운반체(7.8MB ENCRYPTED, 14~15화 스마트워치 스캔으로 첫 확인).
- **오퍼레이터(가칭)**: 스콜피온이 속한 다국적 브로커 네트워크의 상위 조율자. **1권 최종
  빌런 + 차기권 상위 빌런 후보**. "책상 위의 칼". "Ghost 교전 생존자=배신자" 괴담을
  신앙처럼 믿어 폐 PC방 좌표 노출을 근거로 스콜피온을 배신자로 규정 → 뒷치기.
  단 Ghost/정우의 진짜 정체는 **모른다**(누구도 전체 그림을 모르는 정보 비대칭 구조).

## 3. 인물 캐넌 보강 (본문 산출물 + 설정집)

### 정우 — 34세, 과천청사 시설관리 7급 주사보
- 부모를 **약 20여 년 전 어린 시절 사고로 잃음**("아이를 감싸다 부모 사망" 미담성, 단
  의문점 잔존 = 4~5권 떡밥). 정우는 흐릿한 악몽 + "내가 잘못해서" 막연한 죄책감 →
  **2권 초반 악몽씬에서 첫 언급.**
- 누나·매형 사망 후 유나를 떠맡은 생계형 가장. 7년째(설정집 표엔 6~5년 전 임용설도
  병기 — §불일치 참조) B구역 단일 보직.

### 정하린 — 29세, 대테러 4팀장 / 제목의 "국정원 에이스"
- **트라우마 구조(작품 핵심 정서)**: ① 교육센터 "아이 vs 목표" 시뮬레이션에서 교본대로
  아이를 포기("합격") → ② 몇 년 뒤 실전 지휘 라인에서 거의 동일 구조, 실제 아이 사망 →
  ③ 사후 분석 시뮬에서 두 기억이 겹쳐 "아이만 살리는 선택을 검토조차 안 한 사람은 나"로
  굳음. 이후 아이·민간인 얽힌 상황에서 **0.2~0.3초 미세 경직**(= S-1 승급 보류 사유).
- **"선배님" 호칭의 기원(외전3)**: 교육센터 Zero/No.0 패턴 케이스 영상 — "아이를 먼저
  빼내고도 끝까지 임무를 마친 사람"(교본 밖에서 성립, +47초 지연·부상·로그손실 12초).
  정하린은 거기 매료 → 미제출 노트 "언젠가 만나면 '선배님'이라 부르자". 정우=그 Zero로
  오독하며 모든 호칭과 신앙이 출발.
- 자기규정: "문서상 팀장은 나, **진짜 팀장은 선배님**." 정우의 우발 행동까지 "선배님의
  시험/숨은 의도"로 해석하는 확성기. 철학: **"칼은 뽑기 전이 가장 무섭다"**(Ghost를
  실제 병기가 아니라 모두가 오해하는 미지의 리스크로 봉인 운용).
- 외형: 짙은 칼단발, 도시적·차가운 인상, 카리스마(레퍼런스: 후부키 분위기만 차용).
- **2권 메인 POV + 총알 시퀀스**: 정하린에게 날아오는 탄을 정우가 몸으로 대신 맞음(심장
  부근 유나/하린 상징물이 탄도를 비틀어 치명타 회피, 첫 대사 "아, 이거 산재 처리 되냐…")
  → 트라우마가 뒤집히며 일시적 S-1 상위급 각성.

## 4. 권별 구조 — "트라우마 힐링 메타"

각 권: 상처 있는 조연 등장 → 사건으로 상처 벌어짐 → 정우가 **자기식 동기**(칼퇴·징계
회피·내 사람 보호)로 움직임 → 조연의 상처가 덜 아픈 쪽으로 이동 + 정우의 "내 사람"
범위가 넓어짐. 최종적으로 **정우 자신도 구원받는 엔딩**으로 수렴.

- **1권 — 스콜피온 편**: Ghost 탄생 / 행정적 은신 / TF 발족 / 스콜피온 첫 패배+알바 떡밥
  / 오퍼레이터·상위 네트워크 실루엣. 엔딩에 [ID 충돌: JW-0 vs JW-0007] 로그.
- **2권 — 정하린 편**: 하린 메인 POV / JW-0 교육 플래시백 / **총알 시퀀스**로 하린의
  공포·신앙 재정의. 러브라인 씨앗 확실히 심김.
- **3~5권**: 누나(Admin-L)의 BlackArray 조작 / Grey Zone 형성 진실 / "전설적 요원은
  사실 누나였을 가능성"을 서서히 공개.

### 러브라인 — 정우 ↔ 정하린 (메인 축, 매우 느린 장기전)
- 1~2권: 하린 → 정우 = 교본 선배님 신앙+동경 / 정우 → 하린 = "칼 같은 팀장, 내 인사권
  쥔 사람". 2권 총알 시퀀스 이후 하린 쪽에서 신앙과 연애감정이 구분 안 되게 섞이기 시작.
- 3권+: 직접 연애 묘사(데이트·고백) 최소화. 같이 욕먹고 같이 버티는 정서로 쌓음.
- 엔딩: **열린 결말**("선배님 제 마음은…"까지만, 고백 문장은 열어둠). 확정 연애 엔딩 X.
- ⚠️ **유나는 이 라인과 무관**(코미디/정서 축). 러브라인은 오직 정우↔정하린.

## 6. 본문(50화) ↔ 설정집 불일치 / 주의

- **정우 임용 시점**: 본문·주 설정 "7년" vs 연표 표 "6~5년 전" 병기. → 리부트는
  **"7년"으로 통일** 권장(본문 다수·강무영 6화 오독의 근거가 7년).
- **스콜피온 본명**: 본문 "개소문"(27화) = 설정집 "개소문(연개소문에서 연 떼고 줄임)"
  으로 일치 확정. 외전 "17번"은 훈련소 번호(본명과 별개).
- **강태오**: 설정집 메인 로스터에 없음 → **1부 한정 적 캐릭터(JSICS 리더)로 확정**.
  제목의 "국정원 에이스"는 정하린.
- **D-08 정체**: 6권에서 **강무영 원장의 딸**로 공개 예정(1부에선 일반 분석요원으로만).
- **P-03 ↔ C-11**: 친남매(C-11이 오빠, 집안 재무 담당). P-03 모티브 = 엣지러너 레베카.
- **유나 거주**: 본문/설정집 모두 "동거 아님, 사실상 혼자 삶 + 루틴 돌봄"으로 일치.
  (리부트에서 가족 코미디 톤을 살리되 이 생활 구조를 깨지 말 것.)
